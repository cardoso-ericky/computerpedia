Computerpédia

No Computerpédia, encontrarás informações detalhadas sobre software, hardware e malware, explicadas de forma clara e acessível. Nosso objetivo é apresentar o básico do complexo mundo da tecnologia para pessoas leigas, tornando o conhecimento mais acessível.

O nome veio da junção das palavras "Computer" (computador, em inglês), e "Wikipédia" (a maior enclicopédia online da atualidade). Para o software escolhemos o azul escuro, pois achamos que combinava com o tema. Para o hardware, escolhemos o rosa e branco, pelo simples motivo de que todos amam um setup clean e bonito, assim como essas cores. Para o malware escolhemos vermelho, pois remete ao perigo e alerta.

O que é um Software

Software é um conjunto de instruções, programas e dados que permitem que um computador, smartphone ou qualquer dispositivo eletrônico realize tarefas específicas. Diferente do hardware, que é a parte física do dispositivo, o software é intangível, mas podemos usar e interagir com ele.

Os softwares dividem-se principalmente em três tipos: software de sistema, software aplicativo e software de programação.

  • De Sistema:

    • Gerir o hardware e os outros softwares, servindo como base para a execução de programas (Exemplos: Sistemas operativos como Windows, macOS e Android).

  • Aplicativo:

    • Realizar tarefas específicas e atender às necessidades dos utilizadores finais (Exemplos: Microsoft Word, Google Docs, WhatsApp, Zoom.).

  • De Programação:

    • Ferramentas usadas por desenvolvedores para criar outros softwares. (Exemplos: VScode, Git)

O sistema operacional é o programa fundamental que permite que o computador ou celular funcione. Ele controla os recursos do hardware, como memória, processador e armazenamento, e cria a base para que outros programas possam rodar. É também responsável por oferecer uma interface gráfica ou por linha de comando, por onde o usuário interage com a máquina. Windows, macOS, Linux, Android e iOS são exemplos conhecidos, cada um com suas particularidades.

O navegador é um programa desenvolvido especificamente para acessar e interpretar conteúdos da internet. Ele transforma códigos escritos em linguagens como HTML, CSS e JavaScript em páginas que podem ser vistas e utilizadas de forma visual, com textos, imagens, vídeos e botões interativos. Além disso, navegadores modernos são verdadeiras plataformas, capazes de rodar aplicações completas, como editores de texto, serviços de streaming e até jogos online. Chrome, Firefox, Edge e Safari estão entre os mais usados.

Aplicações web são programas que rodam diretamente no navegador e dependem da internet para funcionar. Não é necessário instalar nada no computador, e exemplos comuns são o Gmail, o Google Docs e o Spotify Web. Já os aplicativos móveis são desenvolvidos para rodar em smartphones e tablets, sendo baixados em lojas digitais como a Google Play ou a App Store. Diferentemente das aplicações web, os apps móveis podem aproveitar ao máximo os recursos do dispositivo, como GPS, câmera e sensores de movimento.

Firmware é um software especial gravado dentro do próprio hardware, responsável por dar as instruções básicas para que ele funcione. É o que permite, por exemplo, que um roteador saiba como transmitir internet ou que uma placa-mãe inicialize o computador. Já os drivers são softwares que permitem que o sistema operacional reconheça e utilize corretamente componentes específicos, como uma impressora ou uma placa de vídeo. Sem firmware, o hardware não teria sequer condições de iniciar suas funções. Sem drivers, o sistema não conseguiria conversar com os dispositivos conectados a ele.

Um banco de dados é o software responsável por armazenar, organizar e disponibilizar informações de forma estruturada. É nele que ficam guardados cadastros de usuários, mensagens de aplicativos, fotos de redes sociais e até transações financeiras de bancos. Alguns bancos de dados, como MySQL e PostgreSQL, organizam informações em tabelas relacionais, enquanto outros, como o MongoDB, são não relacionais e oferecem maior flexibilidade.

Bibliotecas são coleções de funções prontas que os programadores utilizam para não precisar reescrever código do zero. Frameworks são estruturas maiores, que fornecem um esqueleto pré-montado para acelerar o desenvolvimento de aplicações, ditando inclusive padrões de organização. Já as APIs funcionam como interfaces que permitem que diferentes softwares conversem entre si, compartilhando recursos e informações de maneira padronizada. Um aplicativo de transporte, por exemplo, usa a API do Google Maps para exibir rotas dentro do próprio sistema.

O GitHub é uma plataforma online voltada para o armazenamento, compartilhamento e colaboração em projetos de software que utilizam o sistema de controle de versão Git. Ele permite que desenvolvedores trabalhem em equipe, criem branches para desenvolver novas funcionalidades, façam pull requests para sugerir mudanças e registrem o histórico completo do código. Além disso, o GitHub oferece integração com ferramentas de automação (CI/CD), gerenciamento de issues, wikis e hospedagem de páginas. É hoje uma das principais redes sociais de programadores, onde milhões de projetos de código aberto e privado estão disponíveis para estudo, contribuição e desenvolvimento.

Sistemas de controle de versão (VCS) são softwares que registram todas as alterações feitas em arquivos de código ou documentos, permitindo que desenvolvedores acompanhem a evolução do projeto ao longo do tempo. Com eles, é possível saber quem fez determinada modificação, comparar versões, recuperar estados anteriores e trabalhar em equipe sem que um sobrescreva o trabalho do outro. O Git é o sistema mais popular atualmente, utilizado em plataformas como GitHub e GitLab. Em um projeto de software, um VCS funciona como um histórico organizado que garante ordem e segurança no desenvolvimento colaborativo.

A Internet das Coisas é a aplicação de software em objetos físicos comuns, conectando-os à internet para que se tornem dispositivos inteligentes. Uma lâmpada controlada pelo celular, um relógio que mede batimentos cardíacos ou uma geladeira que envia alertas sobre alimentos são exemplos. Esses softwares permitem automação, monitoramento remoto e coleta de dados em tempo real, integrando o mundo físico ao digital. Porém, muitos dispositivos IoT têm pouca ou nenhuma proteção, o que os torna alvos fáceis de invasões que podem comprometer não apenas o aparelho, mas toda a rede onde estão conectados.

Inteligência Artificial (IA) é o ramo da computação que busca criar sistemas capazes de executar tarefas que, normalmente, exigiriam inteligência humana. Isso inclui reconhecer imagens, interpretar linguagem natural, tomar decisões e aprender com dados. Hoje, a IA está presente em assistentes virtuais, recomendações de filmes e músicas, carros autônomos e até diagnósticos médicos. Suas técnicas principais incluem aprendizado de máquina, redes neurais e processamento de linguagem natural. Apesar dos benefícios, também levanta debates sobre ética, privacidade e o impacto no mercado de trabalho.

Linguagens de programação são conjuntos de regras e comandos usados para criar softwares. Elas permitem que desenvolvedores instruam o computador a realizar tarefas de maneira lógica e estruturada. Existem linguagens de baixo nível, como Assembly e C, mais próximas do hardware, e linguagens de alto nível, como Python, Java e JavaScript, que são mais fáceis de entender e escrever. Cada linguagem tem pontos fortes: Python é popular em ciência de dados e IA, Java em aplicações corporativas, e JavaScript no desenvolvimento web. A escolha da linguagem depende do tipo de projeto, da performance necessária e do ecossistema de ferramentas disponíveis.

O que é um Hardware

Hardware refere-se à parte física e tangível de um sistema de computador, englobando todos os componentes eletrônicos e mecânicos que o constituem. Inclui dispositivos internos como a placa-mãe e o processador, bem como periféricos externos como o teclado e o monitor. É o lado físico que processa, armazena e transmite dados

A placa-mãe é como o sistema nervoso do computador. Ela é uma placa de circuito impresso que conecta todos os outros componentes, permitindo que eles se comuniquem entre si.

É nela que conecta-se a CPU, a memória RAM, a placa de vídeo, os discos de armazenamento e outros periféricos. É o principal centro de comunicação.

A primeira placa-mãe (na verdade, uma placa de circuito que integrava a CPU) foi criada em 1981, para o primeiro IBM PC. Novos modelos são lançados constantemente, geralmente junto com novas gerações de processadores, como as placas-mãe com chipset Intel Z890 e AMD X880.

O processador, conhecido também como CPU (Central Processing Unit) é o "cérebro" do computador. Ele executa as instruções e realiza os cálculos necessários para que os programas funcionem.

Ele realiza as operações lógicas, aritméticas e de controle. Sua velocidade é medida em gigahertz (GHz) e o número de núcleos (cores) influencia o desempenho em multitarefas.

O primeiro microprocessador comercialmente disponível foi o Intel 4004, lançado em 15 de novembro de 1971. Os processadores estão em constante evolução, a AMD lançou recentemente a série Ryzen 9000 em meados de 2024, e a Intel a série Core Ultra 200 em 2025.

A RAM (Random Access Memory) é uma memória de curto prazo usada para armazenar temporariamente os dados que a CPU está usando no momento. Quando o computador é desligado, a RAM é limpa.

Armazena dados de programas e sistemas operacionais em execução, permitindo acesso rápido para a CPU. Quanto mais RAM, mais programas podem ser abertos simultaneamente sem lentidão.

A memória de acesso aleatório foi desenvolvida por Jay Forrester em 1951, baseada em tubos de vácuo, mas a RAM como a conhecemos hoje começou a ser comercializada a partir de 1968. O primeiro chip DRAM (Dynamic RAM) foi o Intel 1103, lançado em 1970. A tecnologia mais recente disponível no mercado é a DDR5, lançada em 2020, sendo utilizada no Corsair Dominator Platinum RGB DDR5 e no G.Skill Trident Z5 RGB DDR5

A Placa de Vídeo, também conhecida como GPU (Graphics Processing Unit) é um processador especializado em renderizar imagens, vídeos e gráficos 3D. Ela é crucial para jogos, edição de vídeo e programas de design.

Acelera a criação de imagens, transferindo a carga gráfica da CPU.

A primeira placa de vídeo comercialmente disponível foi a IBM Monochrome Display Adapter (MDA), lançada em 1981. No entanto, a primeira GPU a ser chamada de "GPU" foi a NVIDIA GeForce 256, lançada em 1999. No início de 2025 a NVIDIA lançou a série RTX 5000 e a AMD a série Radeon 8000 no final do mesmo ano, sendo as mais recentes do mercado.

É o local onde os dados são armazenados permanentemente, mesmo quando o computador é desligado. Existem dois tipos principais: HDD (disco rígido) e SSD (unidade de estado sólido).

O HDD usa discos magnéticos giratórios para armazenar dados, enquanto o SSD utiliza memória flash, sendo mais rápido e resistente a choques.

O primeiro disco rígido foi o IBM 305 RAMAC, anunciado em 4 de setembro de 1956, já o primeiro SSD comercial foi produzido pela Fusion-io em 2007, mas o conceito já existia desde os anos 70. Novas tecnologias de SSD, como os SSD M.2 PCIE Gen 5.0, são as mais recentes, oferecendo velocidades de leitura e gravação muito superiores às gerações anteriores.

A fonte de alimentação converte a energia elétrica da tomada para os níveis de voltagem corretos que cada componente do computador precisa.

Fornece energia para a placa-mãe, CPU, GPU, discos de armazenamento e outros componentes.

Fontes de alimentação para computadores surgiram com os primeiros computadores pessoais nos anos 70. O padrão ATX (Advanced Technology eXtended), que a maioria das fontes atuais segue, foi criado pela Intel em 1995. As fontes de alimentação atuais mais recentes estão seguindo o padrão ATX 3.0, otimizado para as placas de vídeo mais potentes.

O cooler é responsável por manter os componentes do computador em temperaturas seguras, evitando superaquecimento.

Existem dois principais tipos de refrigeração: a ar (ventiladores e dissipadores de calor) e líquida (watercoolers).

Os primeiros sistemas de refrigeração surgiram junto com os primeiros processadores, quando a necessidade de dissipar calor começou a ser relevante. Hoje, marcas como Noctua, Corsair e Cooler Master são líderes em soluções de resfriamento para PC.

O gabinete é a estrutura que abriga todos os componentes do computador, protegendo-os e organizando-os para circulação de ar e segurança física.

Existem vários tamanhos e formatos, como ATX, Micro-ATX e Mini-ITX, que influenciam na compatibilidade com placas-mãe, placas de vídeo e sistemas de refrigeração.

Os gabinetes modernos incluem recursos como vidro temperado, iluminação RGB e suporte para watercoolers, enquanto os primeiros gabinetes surgiram nos anos 80 como simples caixas metálicas para abrigar componentes básicos.

Periféricos são dispositivos externos conectados ao computador para expandir suas funcionalidades. Exemplos comuns incluem teclado, mouse, monitor, impressora e alto-falantes.

Podem ser classificados em:

  • De entrada: teclado, mouse, scanner
  • De saída: monitor, impressora, alto-falantes
  • De entrada e saída: fones de ouvido com microfone, telas sensíveis ao toque

Com o avanço da tecnologia, surgiram dispositivos mais sofisticados, como mouses ópticos, monitores LCD e impressoras 3D.

O que é um Malware

Malware vem de "malicious software" (“software malicioso”) é qualquer programa criado com a intenção de prejudicar, explorar ou acessar sistemas sem autorização. Não são limitados apenas a computadores: smartphones, tablets, servidores, até dispositivos de IoT (Internet das Coisas) podem ser infectados.

O malware geralmente precisa enganar o usuário para entrar no sistema, ou explorar falhas no software. Alguns métodos comuns de infecção são:

  • Download de programas ou arquivos
  • Abertura de anexos de e-mails ou mensagens suspeitas
  • Exploração de falhas de sistemas
  • Downloads automáticos através de sites maliciosos

Um Trojan, ou cavalo de Troia, é um tipo de malware que se disfarça como um programa legítimo para enganar o usuário. Diferente de vírus tradicionais, ele não se replica sozinho, mas permite que atacantes acessem o sistema de forma oculta. Uma vez instalado, pode roubar dados, instalar outros malwares ou controlar o dispositivo remotamente. Trojans geralmente chegam por downloads suspeitos, e-mails ou links maliciosos.

Um spyware é um tipo de software malicioso projetado para coletar informações do usuário sem seu consentimento. Ele pode registrar atividades no computador, como senhas, histórico de navegação e dados pessoais. Spywares frequentemente se instalam junto com programas gratuitos ou downloads suspeitos. Diferente de vírus, geralmente não danifica o sistema, mas compromete a privacidade.

Um ransomware é um tipo de software malicioso que bloqueia o acesso aos arquivos ou ao sistema do usuário até que um resgate seja pago. Ele geralmente se espalha por e-mails com links ou anexos maliciosos, downloads suspeitos ou vulnerabilidades do sistema. Diferente de outros malwares, seu foco principal é financeiro, exigindo pagamento em criptomoedas. Ransomwares podem criptografar documentos, fotos e bancos de dados, causando prejuízos significativos tanto para usuários domésticos quanto para empresas.

Um worm é um tipo de software malicioso capaz de se replicar e se espalhar automaticamente de um computador para outro, sem precisar de interação do usuário. Eles exploram vulnerabilidades de redes ou sistemas operacionais, causando infecções rápidas e amplas. Diferente de vírus, worms não precisam se anexar a outros programas para se propagar. Embora alguns worms possam apenas sobrecarregar redes, outros podem carregar cargas maliciosas adicionais, como roubo de dados ou instalação de backdoors.

Um adware é um tipo de software que exibe anúncios indesejados no computador ou dispositivo do usuário. Ele geralmente é instalado junto com programas gratuitos ou downloads suspeitos e coleta dados sobre hábitos de navegação para direcionar publicidade. Diferente de vírus ou ransomwares, o adware raramente danifica arquivos, mas compromete a experiência do usuário com pop-ups, redirecionamentos e lentidão no sistema. Em alguns casos, ele pode abrir portas para outros malwares.

Um keylogger é um tipo de malware que registra tudo o que o usuário digita no teclado, incluindo senhas, mensagens e dados pessoais. Ele pode ser instalado através de downloads suspeitos ou anexos de e-mail maliciosos. Diferente de vírus que podem danificar arquivos, o keylogger foca exclusivamente no roubo de informações confidenciais. Esse tipo de malware é usado principalmente para fraudes, roubo de identidade e acesso não autorizado a contas online.

Um downloader é um tipo de malware que baixa outros programas maliciosos para o computador do usuário. Ele normalmente chega junto a arquivos ou programas aparentemente legítimos. Diferente de vírus, o downloader por si só pode não causar danos imediatos, mas atua como um “porta de entrada” para malwares mais perigosos, como ransomware, trojans ou spyware. Seu perigo está justamente em facilitar infecções mais graves e difíceis de remover.

Um rootkit é um tipo de software malicioso projetado para se esconder no sistema e permitir que invasores tenham acesso remoto sem serem detectados. Ele pode modificar arquivos do sistema, ocultar processos e criar portas de entrada para outros malwares. Diferente de vírus e worms, o rootkit não tem efeito imediato visível, tornando sua detecção difícil. Sua principal função é dar controle prolongado do computador ao invasor, comprometendo completamente a segurança e a privacidade do usuário.

Um bootkit é um tipo de malware que se instala no setor de inicialização do computador, carregando-se antes mesmo do sistema operacional. Isso permite que ele permaneça ativo mesmo após reinstalações do sistema. Diferente de outros malwares, o bootkit é muito difícil de remover e pode executar comandos maliciosos de forma silenciosa. Seu objetivo principal é controlar o computador desde o início e abrir caminho para outros malwares ou espionagem contínua.
